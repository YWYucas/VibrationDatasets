# VibrationDatasets
For the 3.2 experiment, we provided 6 datasets for the vibration control of the ZV input shaper. For D1-D2, each dataset contained 500 sampling points, while for D3-D6, each dataset contained 600 sampling points. These data points are obtained from two platforms, i.e., a overhead flexible beam and a oscillating mechanical arm. In these datasets, "freq" and "zeta" are system parameters, representing the undamped natural frequency and damping ratio respectively, while "Y acceleration" is the absolute value of the maximum acceleration in the Y-axis direction at the end of the flexible beam. Furthermore, D7-D8 are open datasets of robotic manipulaters.
For the 3.3 experiment, we provide D3 and D4, which record the optimal parameter combinations corresponding to different configurations of the two platforms, while D1 and D2 are publicly available datasets.
